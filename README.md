
# AppPrimePerformance

Este é um aplicativo Mobile para Fitness, com o intuito de auxiliar nos seus treinos diarios, desenvolvido em Reeact Native.


## Funcionalidades Principais.

- Divisão de treino.
- Divisão grupos musculares.
- Gif para auxiliar nas execuções dos exercícios.
- Explicação detalhada  de cada exercício.


## Stack utilizada

**Front-end:** React Native, Expo, React Navigation, 

**Back-end:** Axios, Rapid API
